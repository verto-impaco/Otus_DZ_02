Программа телефонный справочник

Может: создать контакт, найти контакт, изменить контакт, удалить контакт, вывести все контакты

Как использовать: после запуска программы ввести число от 1 до 6, где 1 - создать контакт, 2 - найти контакт, 3 - изменить контакт, 4 - удалить контакт, 5 - вывести все контакты, 6 - выйти из программы
